编写程序从下面字符串中提取出所有的数据，并按从大到小的顺序依次输出。

vstr = "12;3.45;0.6;7;8;-0.91;234;5.6;78;9"

你的答案
vstr = "12;3.45;0.6;7;8;-0.91;234;5.6;78;9" list1 = vstr.split(';') # 去除分号 list2 = list(list1) # 将字符串转化为列表 # 去掉列表中的引号 def foo(x): if type(x) == str: return float(x) return x; list3 = list(map(foo, list2)) list3.sort() # 列表排序 list3.reverse() # 列表反转 # 把列表中的元素依次输出 for i in list3: print(i)
参考答案

vstrlist = vstr.split(';')

vlist = list( map( eval, vstrlist ) )

orderedlist = sorted(vlist, reverse=True)

print(orderedlist)

请比较Python的列表和元组数据类型，谈谈两者的异同。

你的答案
同：元组和列表同属序列类型，且都可以按照特定顺序存放一组数据 异：列表是可变序列，元组是不可变序列
参考答案

相同点：list和tuple都是序列数据类型，可以存放若干个不同类型的数据，支持序列的元素下标法访问、切片、迭代等操作。

不同点：list是可变类型，数据可以修改；tuple是不可变类型，创建后大小固定，不可修改、添加或删除其中的元素。
